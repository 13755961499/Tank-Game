# 坦克大战 (Tank Game)

复古像素风坦克大战，支持单机与联机合作模式。

- 渲染：HTML5 Canvas
- 客户端：原生 JavaScript
- 联机：Node.js + Socket.io（服务端权威）

## 核心玩法

- 子弹对消：玩家子弹与敌方子弹碰撞会相互抵消
- 敌方类型：普通敌人 / 精英坦克 / BOSS
- 道具系统：击败敌人掉落道具，道具会在地面停留一段时间后消失
- 通关条件：击杀 BOSS 显示“恭喜通关！”
- 联机体验：
  - 至少 2 名玩家才会开始游戏（PLAYING）
  - 服务端统一驱动 AI（移动、射击、生成），客户端只做表现与输入
  - 玩家出生点由服务端分配，避免重叠出生
  - BOSS 血量在 HUD 中显示并实时更新

## 平衡参数（最新）

- 激光伤害：固定 3（对任何敌人一致）
- 精英坦克血量：3
- BOSS 血量：30
- 炸弹对 BOSS：固定扣 5 点
- 敌人射击频率：服务端限制，最低间隔 0.5s
- 道具地面停留时间：10s
- 道具效果持续时间：
  - 星星（攻速）：10s
  - 铲子（基地加固）：30s
  - 护盾：10s

## 运行方式

### 单机模式

直接用浏览器打开 `index.html` 即可开始。

### 联机模式（本地/局域网）

1. 安装依赖
   ```bash
   npm install
   ```
2. 启动服务器
   ```bash
   node server.js
   ```
3. 访问游戏
   - 本机：`http://localhost:3000`
   - 队友：`http://你的局域网IP:3000`（启动日志会输出可用 IP）

```text

├── server.js          # Node.js 服务端逻辑（游戏状态、AI、道具、计分、通关）
├── index.html         # 页面入口
├── css/
│   └── style.css      # UI 样式
└── js/
    ├── config.js      # 配置与平衡参数
    ├── audio.js       # 音效
    ├── sprite.js      # 绘制
    ├── map.js         # 地图与碰撞
    ├── tank.js        # 坦克与射击（含激光）
    ├── powerup.js     # 道具实体（单机模式停留时间）
    └── game.js        # 主循环、联机同步、HUD
```
